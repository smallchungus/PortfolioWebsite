# ADR-001: Design Philosophy for 2025

## Status
Accepted

## Context
Modern portfolio designs in 2025 favor minimal brutalist aesthetics over flashy animations. Reference sites like Linear.app, Stripe.com, julioelpoeta.com, and leerob.io demonstrate clean typography-focused designs that prioritize content over decoration.

## Decision
Implement minimal brutalist design philosophy:
- **Typography-first**: Primary focus on hierarchy and readability
- **Monochrome base**: Black/white/gray with single accent color (#2563eb)
- **No decorative elements**: Remove all unnecessary visual flourishes
- **Content hierarchy**: Clear information architecture over visual complexity
- **Performance-first**: Zero decorative animations affecting load time

## Consequences
**Positive:**
- Faster load times and better Core Web Vitals
- Timeless design that won't look dated
- Better accessibility and readability
- Easier maintenance and content updates
- Professional appearance for FAANG-level positions

**Negative:**
- May appear generic compared to highly stylized portfolios
- Less opportunity to showcase visual design skills
- Requires excellent content to stand out

## References
- Linear.app: Clean typography, minimal color palette
- Stripe.com: Content-first approach, subtle interactions
- leerob.io: Developer-focused minimal design
- julioelpoeta.com: Brutalist portfolio execution
